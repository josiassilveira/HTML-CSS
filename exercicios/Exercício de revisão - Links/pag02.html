Revisando os Links

Links Internos

Já os liks internos servem mais para optimizar a transição das páginas dentro de seu site ou para abrir links de seu próprio programa.


Para isso, basta criar um novo arquivo em sua pasta indicando a página que deseja. Com isso, adicionar a tag <a> depois colocar também as tags "target" e "rel" mudando apenas as informação que vão dentro deles tendo como alvo os arquivos dentro de seu site ou programa.
