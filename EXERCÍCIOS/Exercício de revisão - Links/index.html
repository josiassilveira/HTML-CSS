Links Externos

Existem os links externos que são URLs de fora do seu site. E para criar um atalho para ele basta colocar uma frase dentro da tag <a> e depois colocar também as tags "target" e "rel"

Ex: Para acessar um perfil de lo-fi no YouTube basta clicar aqui.Só abrir o VS Code e ver como está.
