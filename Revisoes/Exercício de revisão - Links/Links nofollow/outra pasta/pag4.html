Revisando os Links

Páginas dentro de pastas

Para acessar uma página dentro de uma pasta deve-se apenas seguir o mesmo processo dos outros links das paginas anteriores, mas o que diferencia é que se deve criar um arquivo "html" antes para que ela possa funcionar.

Retorno de páginas

Para retornar a página anterior sendo que a atual está dentro de uma pasta, basta apenas colocar dois pontos e uma barra "../" e em seguida colocar o link da página que deseja retornar (isso se ela estiver fora dessa pasta).
