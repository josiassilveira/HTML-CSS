Revisão Básica

Primeira foi mostrado sobre a questão dos títulos e também sobre os parágrafos. Foi mostrado que para criar um título é necessário usar a fórmula "<"h1 e para fechá-lo é necssário usar ">h1". Já para abrir um parágrafo é necessário usar a fórmula "<p" e para fechá-la é preciso usar a fórmula >p.

Também foi explicado a criação de linhas mas a princípio sobre a linha horizontal que pode ser criada para fazer separação entre o títuo e o resto do texto. Usando a fórmula "<hr" faz com que essa linha seja criada. Falando sobre esses tipos de fórmulas que criámos para deixar o nosso texto com uma "cara melhor", existe também a fórmula "<br" que serve para fazer uma quebra de linha

Agora, se você quer deixar o seu site ou texto mais bonito, existe fórmulas e meios para isso. Cada forma tem um nome em específico mas aqui vai aguns exemplos:
Δ
®
™
↓
∅
Caso queira saber mais, ecessar o site Misc Symbols.
Existem também certas figuras chamadas de emojis. Para fazer esses emojis é necessário pesquisar o seu código em sites e o mais recomendado é o "emojipedia.org". Depois que foi pesquisado, para fazer com que eles aparecem em seu texto, é preciso colocar a fórmula "&#x". Exemplos:
"& #x1F60A" (e não esquercer de colocar o sinal de ";"no final da fórmula) → 😊
"& #x1F4A5" (e não esquercer de colocar o sinal de ";"no final da fórmula) → 💥
"& #x26F0" (e não esquercer de colocar o sinal de ";"no final da fórmula) → ⛰

Vamos falar também de imagens, elas que são de grande importância tanto para livros como para sites, trabalhos, etc, já que são elas que fazem o "chama" das pessoas. Mas é muito importante tomar muit- cuidaod com as figuras que você programador coloca em seu site pois ela pode conter direitos autorais ©.
É muito importante analizar bastante antes de colocá-las a amostra; é aconselhável que se crie imagens ou que elas sejam extraídas de sites onde permitem o uso livre delas para que não possa gerar nenhum tipo de problema futuro.
Há certos tipos de classificação de imagens sendo elas:

JPG → gera arquivos muito pequenos e que ocupam pouco espaço no disco. Posso também escolher quanto de compactação será exibido. IMPORTANTE: De forma resumida, na hora de escolher o formato de imagem para o seu site, dê preferência ao formato JPEG com uma compactação entre 30% e 50%. O formato PNG só deverá ser usado quandp precisarmos de transpência na foto.
PNG → capacidade de configurar a opacidade de cada pixel (deixá-lo transparente ou com transpência limitada).
